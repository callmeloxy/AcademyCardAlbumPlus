# AcademyCardAlbum+

Academy-style paginated card album mod for Minecraft `1.21.1`.

## Overview

AcademyCardAlbum+ is a storage-focused utility mod designed to provide an Academy-style paginated card album experience for Minecraft `1.21.1`.

It focuses on clean album behavior, page-based storage, and a user experience aligned with the original Academy album style.

## Features

- Academy-style paginated card album
- Page-based storage system
- Clean storage-focused user experience
- Configurable album page count
- Configurable overflow behavior
- Support for Minecraft `1.21.1`
