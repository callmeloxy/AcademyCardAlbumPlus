# AcademyCardAlbum+

Mod d’album de cartes paginé dans le style Academy pour Minecraft `1.21.1`.

## Aperçu

AcademyCardAlbum+ est un mod utilitaire orienté stockage, conçu pour proposer une expérience d’album de cartes paginé dans l’esprit de l’album Academy sur Minecraft `1.21.1`.

Le projet met l’accent sur un comportement propre, un stockage par pages, et une expérience utilisateur cohérente avec le style de l’album Academy original.

## Fonctionnalités

- Album de cartes paginé dans le style Academy
- Système de stockage par pages
- Expérience utilisateur propre et orientée stockage
- Nombre de pages configurable
- Gestion du dépassement configurable
- Support de Minecraft `1.21.1`
